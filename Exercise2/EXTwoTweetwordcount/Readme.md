# Exercise 2
### Twitter stream parsing



## Instructions

1. Initialize environment with start.sh
  * Must be set to executable: `chmod +x start.sh`
 
2. Run create_database.py
  * This script will create both the database and the *wordcount* table.
 
3. `cd /EXTwoTweetwordcount`

4. `sparse run`
  * This is not a timer-limited application.  Kill the process after a few minutes.
